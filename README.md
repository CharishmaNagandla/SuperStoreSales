# Superstore Sales Analysis Dashboard – Power BI

## Project Overview
This project analyzes retail sales data from a Superstore dataset using Microsoft Power BI. The goal is to transform raw transactional data into an interactive dashboard that provides insights into sales performance, product trends, customer behavior, and regional performance.

---

## Business Problem
Retail businesses generate large amounts of transactional data, but it can be difficult to identify sales trends, top-performing products, and valuable customers without proper visualization tools.

This project aims to solve this by building a Power BI dashboard that allows stakeholders to easily monitor sales performance and identify growth opportunities.

---

## Project Objectives
The goal of this project is to build an interactive Power BI dashboard that:

- Provides a clear overview of overall business performance
- Identifies top-performing product categories and items
- Analyzes customer segments and top customers
- Examines geographic sales distribution across states and cities
- Enables users to interactively explore insights through filters and visuals

---

## Dataset Information
Dataset: Superstore Sales Dataset

Main fields include:
- Order Date
- Customer Name
- Segment
- Product Category
- Sub-Category
- Sales
- Quantity
- State and City

Dataset Source: Kaggle
